
# ☕ Coffee Shop Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes coffee shop sales data using Microsoft Excel to generate actionable business insights through KPIs, charts, slicers, and interactive dashboards.

---

## 🎯 Objectives
- Analyze sales trends across weekdays and hours
- Identify peak customer activity periods
- Compare sales performance across store locations
- Analyze top-selling products and categories
- Track customer ordering behavior
- Build an interactive Excel dashboard

---

## 🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Cards
- Slicers & Filters
- Data Cleaning
- Dashboard Design
- Data Visualization

---

## 💡 Key Insights
- Morning hours generated the highest sales activity.
- Coffee products contributed the highest revenue share.
- Hell’s Kitchen recorded the highest sales among all locations.
- Barista Espresso was the top-selling product.
- Weekdays experienced higher customer transactions compared to weekends.
